Module @open-pioneer/chakra-integration - v2.3.0

@open-pioneer/chakra-integration

This package integrates Chakra UI into an Open Pioneer Trails application. All components are re-exported from @chakra-ui/react (sometimes with some modifications). All UI packages using the Open Pioneer Trails framework should use this package instead of depending on @chakra-ui/react directly.

Internals

Some changes are made to complex components such as Modal, Drawer etc. to support integration into a web component's shadow DOM. For more details, see comments in ./Provider.tsx.
